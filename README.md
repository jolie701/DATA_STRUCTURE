# Data Structures Course Assignments

This repository contains all the programming assignments and analysis reports completed during the Data Structures course. It covers core concepts including sorting algorithms, fundamental data structures (stacks, queues), and empirical performance testing.

---

## Assignment Directory

| Assignment | Topic | Description | Files Included |
| :--- | :--- | :--- | :--- |
| **HW1** | Insertion Sort & Merge Sort | Implementation and comparative analysis of Insertion Sort and Merge Sort. | Source Code / Documents |
| **HW2** | Stack & Queue | Parentheses balance checker using a stack to validate symmetrical brackets and locate mismatches. | `data_structure-hw2-Stack&Queue.zip` |
| **HW3** | Heap Sort & Quicksort | Implementation of advanced sorting techniques: Heap Sort and Quicksort. | `data_structure-hw3-Heap Sort&Quicksort.zip` |
| **HW4** | Computational Tests & Analysis | Performance testing and empirical analysis across 7 sorting algorithms using custom timers. | `data_structure-hw4-Analysis for Sortiong Algo...` |

## Project Structure Notes

* **Compressed Archives (.zip):** Some assignments are uploaded as `.zip` packages containing the full source codes (`.cpp` / `.c`) and the required execution/output reports (`.pdf` / `.txt`).
* **Source Folders:** Standard directories contain uncompressed source files and materials for immediate review.
